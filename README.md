# Easy Useful Finds

Link-in-bio landing page for @easyusefulfinds.

## GitHub Pages setup

1. Push this repo to GitHub.
2. Open GitHub repo Settings.
3. Go to Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Save.

Public URL should become:

https://gotenlica.github.io/easyusefulfinds/

## Updating products

Edit `index.html` and replace a product button `href="#"` with the Amazon Associates/SiteStripe affiliate link.

Keep the affiliate disclosure visible:

As an Amazon Associate I earn from qualifying purchases.
